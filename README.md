# 🤖 Gemini AI Assistant

An AI-powered assistant built with **Streamlit** and **Google Gemini 2.5 Flash**.

---

## 📌 Features

- 💬 **Text Chat** – Ask questions and get intelligent responses  
- 🖼️ **Image Analysis** – Upload and analyze images with Gemini  
- 🌍 **Multi‑language** – Supports both English and Arabic  
- 💾 **Conversation History** – Save, load, and export chats  
- ⚡ **Fast & Responsive** – Powered by Gemini 2.5 Flash  

---

## 🛠️ Tech Stack

- **Python** 3.9+
- **Streamlit** – UI Framework
- **Google Gen AI SDK** – Gemini API
- **Pillow** – Image processing
